### 描述
**GET** `/v4/balance`

---

### 请求参数

| 参数名     | 类型   | 必填 | 默认值 | 描述   | 取值范围 |
|------------|--------|------|--------|--------|----------|
| currency   | string | 是   | N/A    | 币种，例如：usdt |  |

---

### 返回示例

```json
{
  "rc": 0,
  "mc": "string",
  "ma": [
    {}
  ],
  "result": {
    "currency": "usdt",          // 币种
    "currencyId": 0,             // 币种ID
    "frozenAmount": 0,           // 冻结数量
    "availableAmount": 0,        // 可用数量
    "totalAmount": 0,            // 总数量
    "convertBtcAmount": 0        // 折合BTC数量
  }
}
