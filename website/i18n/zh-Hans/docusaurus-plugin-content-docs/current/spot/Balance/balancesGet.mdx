### 描述
**GET** `/v4/balances`

---

### 限流规则
- 10次/秒/apikey

---

### 请求参数

| 参数名      | 类型   | 是否必填 | 默认值 | 描述                               | 取值范围 |
|-------------|--------|----------|--------|------------------------------------|----------|
| currencies  | string | 否       | N/A    | 币种列表，逗号分隔，例如：usdt,btc |          |

---

### 返回示例

```json
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {
    "totalBtcAmount": 0,
    "assets": [
      {
        "currency": "string",        // 币种
        "currencyId": 0,             // 币种ID
        "frozenAmount": 0,           // 冻结数量
        "availableAmount": 0,        // 可用数量
        "totalAmount": 0,            // 总数量
        "convertBtcAmount": 0        // 折算BTC数量
      }
    ]
  }
}
