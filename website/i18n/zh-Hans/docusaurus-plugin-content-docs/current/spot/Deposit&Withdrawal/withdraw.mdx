### 接口说明
**POST** `/v4/withdraw`

---

### 注意事项
- 参数需以 **JSON** 格式放在 **body** 中。

---

### 限流规则
- 1次/秒/apikey

---

### 请求参数

| 参数名        | 类型   | 是否必填 | 默认值 | 描述                                                                 | 取值范围 |
|---------------|--------|----------|--------|----------------------------------------------------------------------|----------|
| currency      | string | 是       | —      | 币种名称，可通过 `获取支持充提的币种信息` 接口获取                     |          |
| chain         | string | 否       | —      | 转账网络，可通过 `获取支持充提的币种信息` 接口获取                     |          |
| clientOrderId | string | 否       | —      | 自定义订单号，正则：`^[a-zA-Z0-9_]{4,32}$`                           |          |
| amount        | number | 是       | —      | 提币数量（包含手续费）                                               |          |
| address       | string | 否       | —      | 提币地址                                                             |          |
| memo          | string | 否       | —      | 备注，部分链（如 EOS）必须填写                                        |          |
| toAccountId   | number | 否       | —      | 接收用户 ID                                                          |          |

---

### 请求示例

```json
{
  "currency": "zb",
  "chain": "Ethereum",
  "amount": 1000,
  "address": "0xfa3abfa50eb2006f5be7831658b17aca240d8526",
  "memo": ""
}
