### 请求

- **格式:** `kline@{symbol},{interval}`
- **时间间隔:** 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w, 1M
- **示例:** `kline@btc_usdt,5m`
- **频率:** 1000ms

### 响应示例

```json title="响应"
{
  "topic": "kline",
  "event": "kline@btc_usdt,5m",
  "data": {
    "s": "btc_usdt",
    "t": 1656043200000,
    "i": "5m",
    "o": "44000",
    "c": "50000",
    "h": "52000",
    "l": "36000",
    "q": "34.2",
    "v": "230000"
  }
}
```
