**类型：** WEBSOCKET


**描述：** 用于通过 WebSocket 执行主题**订阅/退订**的请求消息结构。


### 参数


| 名称 | 类型 | 是否必填 | 默认值 | 说明 | 取值范围 |
| ------ | ---------------- | -------- | ------ | ------------------------------------------ | ---------------------- |
| method | string | 是 | - | 要执行的操作。 | subscribe；unsubscribe |
| params | array| 是 | - | 主题参数项，如 `{topic}@{arg}`，可多项。 | `"{topic}@{arg}"` |
| id | string | number | 否 | - | 客户端自定义的**回调 ID**，用于响应关联。 | - |


---


### 请求示例


```javascript title="subscribe"
{
"method": "subscribe",
"params": [
"{topic}@{arg},{arg}",
"{topic}@{arg}"
],
"id": "{id}" // 回调ID
}

{
"method": "unsubscribe",
"params": [
"{topic}@{arg},{arg}"
],
"id": "{id}" // 回调ID
}
```

### 请响应示例
```json title="Response"
{
"rc": 0,
"mc": "SUCCESS",
"result": "SUBSCRIBED",
"id": "{id}"
}

```
