title: 查询用户的 API 密钥（适用于主账户和子账户）
sidebar_label: 查询用户的 API 密钥
sidebar_position: 1
---

**类型：** GET

**描述：** `/v4/user/account/api-key`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| userAccountId | string | false | - | 账户 ID | - |
| keyName | string | false | - | API 密钥名称 | - |
| isLock | integer | false | - | API 密钥是否锁定 | 0: 未锁定；1: 已锁定 |

---
